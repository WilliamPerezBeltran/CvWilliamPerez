# CV William Pérez

## Stack

- html
- css
- javascript
- react

### Clone repo and instalation

```bash
# Clone repo
$ git clone https://github.com/WilliamPerezBeltran/CvWilliamPerez.git

# Go to app directory
$ cd CvWilliamPerez

# Install depedencies
$ yarn install or npm install
```

### Run app

```bash
# Run the app in dev mode for linux
$ yarn electron-dev or npm run electron-dev
```
